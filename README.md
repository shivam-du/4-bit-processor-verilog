# Design and Verification of a 4-Bit Processor Using Verilog HDL

## Overview

This project presents the RTL design and functional verification of a custom 4-bit single-cycle processor developed using Verilog HDL and AMD Vivado 2025.2.

The processor architecture consists of:

- Program Counter (PC)
- Instruction Memory (IM)
- Instruction Decoder (ID)
- Control Unit (CU)
- Register File (RF)
- Arithmetic Logic Unit (ALU)

The processor executes custom 8-bit instructions and supports arithmetic, logical and data transfer operations using a simplified Instruction Set Architecture (ISA).

## Features

- 4-bit data path
- 8-bit instruction format
- Single-cycle architecture
- Custom ISA
- Functional verification using simulation
- Modular RTL design

## Development Environment

| Parameter | Value |
